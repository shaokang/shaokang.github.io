Fuck2014-First2015

写在前面

上一年的，搬上来啦，好重唉。

2014年总结

2014年，粗暴点说，过得好快，纵然很多方面都没有按期的实现，没有从一个渣渣变成只大牛，但也算开窍了。技术方面：android应用层掌握的不错，java底子打的更深入了，对几个软件设计模式有基本了解。生活方面：锻炼更少了，放假时候更喜欢懒床了，上班，上课，回宿舍，偶尔聚聚会，打打球，跑跑步，就酱紫，无波无澜。

成果

  1. 2013年年底正式下海，学习android学习，摸着石头过河，狂买书，狂在网上寻找相应资源，跌跌撞撞，自己捣鼓出来几个小demo，工作项目这边10月份上线了个功能阉割版的app，现在健壮版也完成的差不多了，再忍忍赶赶一段时间久可以出来了。这段时间想想，也终于爬出来坑了。
  2. 业余时间，少打游戏很多，抽空，也用kindle看了几本书，会经常留意下互联网界的信息，逛知乎，逛果壳，看技术文章。
  3. 学习方法上，也算摸索出来属于自己的学习方法，也养成解决问题的习惯，学会上，stackoverflow 上寻找答案，github上寻找资源，开发过程中遇到些常见的问题，也心有成竹的done掉。
  4. 英文阅读能力上，提高了不少，指法也飘逸了不少。
  5. 学校学习这边，收获甚少，虽最后抱大腿，得以连续两个学期没有挂科，最后成果也不知道怎样描述。

不足

  1. 对android一些没有用到的部分技术较弱，没有用时间去深入的学习。主业的android 这方向感觉就一个不上不下的地步，迷茫，都不知道自己接下来该专攻哪方面。
  2. 对知识不善于总结，记录，学习线路很乱，（Linux，javaweb，html5，编程基础，jvm,设计模式，手游开发）等等这些都想学习。最后没有一条真正的走下去，倒是迷失了自己。
  3. 代码写得还不是很规范，一直想写技术blog，github上逛了大半年，却没有真正的去学习git使用，最近时间才开始重视git使用。
  4. 没有加入到开源圈子讨论中，惧怕与大牛们交流，心理总是暗示，等自己再厉害点，才敢去找他们（错误心理），没有在git上贡献过代码。

意外收获

  1. 有与工作环境关系，对产品倒了解了不少，掌握了某些产品原型工具。
  2. 对团队、团队管理、创业等等这些词语有了更加直观了理解与体验。

2015展望

知识深入

  1. android主业技术方面，将自己薄弱的几块啃了。重点：深入研究自定义控件，能达到随心所欲的状态。
  2. 学习MaterialDesign方面，并能够运用到自己项目中，或者demo中
  3. 研究hybird app，学习html5，可以简单的进行开发
  4. 选择1——2个比较常用的android开源框架进行研究

知识扩展

  1. JavaWeb（springMVC hibernate)
  2. mySql数据库
  3. linux服务器（需求满足自己毕业设计，毕业要设计尽量拿优秀，弥补一个学渣大学遗憾
  4. 英文阅读能力要提高，专业词汇量要增加
