再见2015－生活篇

发表于 2016-02-18   |   分类于 随笔

再见2015－生活篇

离职

​ 离开一个自己待了一年多的创业团队，这段日子里面日子从一个懵懂的大二学生第一次接触到创业这个词语，那是整天都唱着追梦赤子心的日子，然而在外面的大“世界”中，自己的“世界”还是太小了，所以自己选择出来看看。

找工作

技术素质偏低

​ 大一、大二期间学习不太重视，花了很多时间在社团的，以及乱点“技能树”；然而到大三的某个夜晚才想起自己要出社会了，得重视自己的技术素养了。然而之后的路程也走的比较坎坷，自我的控制不太好，很多时间都与我亲爱的舍友们一起奔跑在召唤师峡谷中，享受着大招放空的喜悦。

​ 加入创业团队中之后，自己学习的过程也是比较坎坷，又想着学习PHP，然后好些日子搭建好开发环境之后，写了个hello wrold，看了点语法之后，又放弃了，然后不断在走弯路。总结出来：入门者加入创业团队，对自己的技术成长会比较坎坷，因为在这样的环境没有时间让你成长，高压会导致自己的不断走弯路，技术基础不扎实。好吧，这个坑自己的也跳了。

忧虑、紧张

​ 对于应届毕业生来讲校招的成果是最具攀比性了，这也是对于大学这四年的一个证明，薪金、公司规模、职位等等。

​ 同学都纷纷拿到了offer了，BAT都有，还有一些其他的大公司等等。3月份的时候，这场互联网校招的战争已经结束了，剩下的也就是一个技术素养不是很好的筒子们在捡漏，当然这里面也包括自己了。4399虽然在我们的圈子里面的评价不太好，但是对于自己来讲，这已经很满足了，然后没有做任何准备就过去了，结果是没有问几个问题就被告知回去等通知。有米，这是第一次尝试叫同学内推，结果也一样，自己没有控制好，老是打游戏，没有做准备，然后一面也是跪了。“摔”你特么就不知道反省下么，然后还有一些公司在笔试的时候就跪了。

​ 班里拿到offer的同学们，也会到处“宣扬”自己的“成果”，这让自尊心强且脆弱的自己，只能通过每晚跑内环的发泄，然后自己并没有瘦下来，大概自己的在跑步的同时也吃多了吧。

​ 缺乏目标、缺乏控制与策略是自己这段求职经历如此坎坷的主要原因。

错过与最后

​ 最后找到了一家H5游戏为主的公司，福利一般，尼玛，终于有人收了自己，校招这个关卡就这样过了。

毕业

​ 毕业了，交毕业设计、拍毕业照、领毕业证、找房子搬出去、滚蛋。

社会、工作

一个人的生活

​ 这么多年都是集体的生活，开始一个人的生活了，改变了很多。开始的时候自己宁愿在公司加班待到十一点也总比回到“家”之后自己的一个人发呆好。引用同事的一句话”在公司中好歹也有一群人陪啊，回到家就真的只有一个人了“，开始的一段时间，心理有无数的东西想跟之前的小伙伴们分享，然后睡醒又开始挤地铁，上班，各自过着各自的生活。

​ 更加珍惜自己的家人、朋友、基友，因为你发现他们是那样珍贵，失去就没了。

​ 意识到自己的要做”自己“，社会上上很多事情都跟之前的不一样了，自己需要更加主动的表达自己的想法，自己之前脸皮比较薄；还有你得去忽略一些事情、更加重视一些事情、懂得独立思考、自我判断能力，做那一个独一无二的自己，做一只“温柔可爱的刺猬”。自己为什么要用到“温柔可爱的刺猬”这个词语呢？懂得怎样去保护自己这点很重要，但是你得心态得是乐观、热爱你的生活、去享受自己的生活。

​ 笑点变低了，有时候，坐着坐着地铁突然心理就冒出来一个诡异的“可笑的话题”，然后一个人就莫名奇妙的笑起来了，这种笑从心里开始蔓延，一只到脸部，然后自己意识到自己又不能大笑起来，然后又尽量让自己的不笑起来，免得别人看起来觉得我很奇怪，好吧，保持这样的动作让自己很吃力，场景脑部吧。这大概是一个人待久了就有一定概率会触发的事情了。又或者是程序员的“特别”的减压方式。

经济独立了

​ 有了收入，能够自己支配自己的经济，想买点啥也有点小钱了，花钱也不知道节制，摔，你特么不知道自己理财么，工作这么久了，一点银两也没有留下来。

​ 第一次给钱嘛嘛，给家人。这样感觉很难描述，对于自己来讲这是十分高兴的一件事情，自己终于可以去“回馈”了，而不是去索取了。还记得刚上大学的时候，嘛嘛问我说：“还够钱用么？”，然后想掏出3百大洋给自己，那时候她在家附近的一个小工厂里面从事劳动密集型产业的工作，一个月也就七八百，最后自己没有拿，因为爸爸已经给自己“伙食费”了。这段经历很犹新，还有姐姐们，刚结婚、工作后给送自己手机，带自己去吃肉的经历等。反正能够有这样的机会去做这样的事情，我觉得很高兴就是了，情感方面我不太擅长表达。

日常

生活习惯也不断转变，大概有这几个阶段

  1. 听着追梦赤子心，每天加班到十点多，早上第一个到，回去还练肌肉，睡前看thinking in java ，复习算法，休息日去白云山奔跑的鸡血满满的生活。
  2. 正常上班，回去打游戏，睡觉，上班，打游戏的略有带颓废的生活。
  3. 正常上班，回家后自己做个小菜，偶尔健健身，追剧，写点东西来纪录自己生活，技术成长，睡前听个小曲略有惬意的生活。

2015年就这样过去啦，time is short!
